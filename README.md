# 100 Days of VLSI – Digital RTL Design

I am documenting my journey to become a digital VLSI engineer.  
This repository contains daily learning notes and complete projects in Verilog.

## 📁 Repository Structure

- `/day01` to `/day16` – daily theory notes and small code snippets
- `/projects` – standalone, well‑documented Verilog projects (each with its own HTML page)

## 🚀 Featured Projects

| Project | Description | Live Page |
|---------|-------------|-----------|
| **Parameterized Synchronous FIFO** | Configurable depth/width FIFO with full/empty/almost flags | [fifo_project.html](fifo_project.html) |
| **3‑Stage Pipelined 8‑bit ALU** | Add, sub, and, or, xor; three pipeline stages | [pipelined_alu_project.html](pipelined_alu_project.html) |

More projects (asynchronous FIFO, UART, RISC CPU) will be added soon.

## 🛠️ Tools Used

- **Icarus Verilog** + **GTKWave** – simulation and waveform viewing
- **EDA Playground** – online simulation (alternative)
- **GitHub Pages** – portfolio hosting

## 📈 Daily Progress
